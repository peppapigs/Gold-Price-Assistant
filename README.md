# Gold Price Assistant (黄金价格监控助手)

这是一个**实时的黄金和白银价格监控看板**。它提供了一个直观的 HTML 网页浏览器界面，让你随时关注国际、国内贵金属走势，并计算投资盈亏。

此外，本项目**原生支持 MCP (Model Context Protocol)**，你可以极其方便地将它作为一个 Tool 插件集成到 Cursor、Claude Desktop 等 AI 客户端中，让 AI 也能实时帮你盯着金价。

---

## 🌟 核心功能

### 1. 网页端实时监控看板 (`gold_monitor.html`)
- **汇率与报盘**：展示美元兑人民币最新汇率，以及国际、国内黄金/白银的实时价格。
- **内外盘价差**：自动计算国内溢价（内外盘价差）。
- **盈亏平衡计算器**：只需要输入你的买入均价、克数及手续费，系统会自动计算盈亏平衡点并实时展示总体盈亏情况。
- **自定义刷新**：支持下拉选择 5秒~60秒 的自定义获取频率。

### 2. MCP Server 插件集成
支持作为 AI 的外部工具链，提供以下能力：
- `get-gold-prices`: 获取实时的国际和国内黄金白银价格的原始数据。
- `get-gold-page`: 直接由 MCP 服务器把完整的监控页面 HTML 代码通过对话返回给 AI 或用户。

### 3. 本地独立数据代理
启动时会自动在 `8083` 端口开放 `/api/gold` 接口。

---

## 🚀 方式一：作为普通网页看板使用（推荐日常看盘）

如果你只是想个人在电脑上打开网页看金价，请按照此步骤操作。

### 1. 环境准备与安装
请确保电脑已安装 [Node.js](https://nodejs.org/) (推荐 v18+)。打开命令行进入本项目根目录，运行：
```bash
npm install
```

### 2. 启动本地代理服务
我们需要先启动本地数据代理服务才能正常拉取数据：
```bash
npm run test:http
```
*(这会同时启动 MCP 的 8080 测试端口和 8083 代理端口，只需保持此命令行窗口不关闭即可)*

### 3. 浏览器查看
**直接双击打开项目中的 `gold_monitor.html` 文件。**
在网页右上角点击【开启获取数据】，看板就会开始自动拉取并刷新实时金价了！

---

## 🤖 方式二：作为 MCP Server 接入 AI 使用（面向开发者/AI 工具）

如果你希望将此项目交给 Cursor 或 Claude Desktop 等大模型客户端使用，使其可以通过此项目获取实时金价数据：

本工具支持标准的 **Stdio 模式** 通信。你无需手动敲击控制台命令，只需按如下步骤：

1. 确保已运行过一次 `npm install` 与 `npm run build`。
2. 在你的 AI 客户端（如 Claude Desktop 的 `claude_desktop_config.json` 或 Cursor 的 MCP 配置界面）中，添加新的 MCP Server：
   - **Command**: `node`
   - **Args**: `["你的项目绝对路径/build/index.js"]`

配置保存后，AI 客户端将在后台自动唤醒此项目。从此你便可以直接在对话框中向 AI 询问：“*帮我查一下现在的国内黄金实时价格*”。
